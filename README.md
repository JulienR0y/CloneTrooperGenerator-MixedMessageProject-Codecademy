## General info

This project is a Clone Trooper ID generator.

Description : This project aim at creating a Clone Trooper ID generator, it will :
- Give you a ID Number,
- Rank,
- Division,
- The last Jedi you fought with,
- The Last planet you fought on,

- JS. ES6+
